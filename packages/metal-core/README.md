# @ferroscale/metal-core

Shared calculation package used by:
1. Ferroscale web app
2. Ferroscale Raycast extension

## What It Contains

1. Calculator engine and validation
2. Profile and material datasets
3. Quick query parser and quick-weight APIs

## Run Tests

From repo root:

```bash
npm run test:core
```

From this folder:

```bash
npm run test
```
